Ce fichier est pour PhonologicalCodesGraphemes. Pour chaque code phonologique (par exemple, % pour /ɑ̃/ dans le mot bilan), le fichier indique le phonème correspondant (en notation API), ainsi que les mots contenant les différents graphèmes (mis en évidence en rouge) correspondant à ce phonème. Les voyelles sont dans des cellules bleues; les consonnes, dans des cellules vertes; les séquences de phonèmes, dans des cellules violettes. Les mots sont triés par fréquence du graphème mis en évidence, estimée par le nombre de mots différents qui le contiennent dans Silex (Lexique + Manulex).

Des couleurs et des filtres sont disponibles en utilisant le générateur de tableaux SILEX original. Vous pouvez sélectionner les codes pour lesquels vous souhaitez des explications en utilisant les filtres Excel fournis. Dans le guide de l'utilisateur, vous trouverez plus de détails sur le codage phonologique et la segmentation graphémique, ainsi que la reproduction exacte de ce tableau, dans l'Annexe A.

This file is for PhonologicalCodesGraphemes. For each phonological code (e.g., % for /ɑ̃/ in the word bilan), the file indicates the corresponding phoneme (in IPA notation), as well as words with the different graphemes (highlighted in red) corresponding to this phoneme. Vowels are in blue cells; consonants, in green cells; sequences of phonemes, in purple cells. The words are sorted by the frequency of the highlighted grapheme, as estimated by the number of different words that contain it in Silex (Lexique + Manulex). 

Colours and filters are available using the original SILEX Table Generator. You can select the codes(s) for which you want explanations by using the Excel filters provided. In the User Guide, you can find more details about phonological coding and graphemic segmentation, as well as the exact reproduction of this table, in Appendix A.
